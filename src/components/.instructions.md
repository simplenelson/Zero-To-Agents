# Component Guidelines

- Use `"use client"` only when the component needs interactivity (event handlers, hooks, browser APIs)
- Accept props via a typed interface — define it in the same file
- Use Tailwind CSS utility classes for all styling
- Follow the existing dark theme: gray-900/950 backgrounds, gray-100/200 text, blue-400/500 accents
- Keep components focused on one responsibility
- Use semantic HTML elements (nav, main, article, section)
